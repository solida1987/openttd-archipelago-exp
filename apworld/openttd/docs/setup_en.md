# OpenTTD Archipelago Setup Guide

## Requirements
- OpenTTD 15.2 (patched with Archipelago integration)
- Archipelago 0.6.0 or later

## Installation
1. Download and install the patched `openttd.exe` from your Archipelago build.
2. Place the `.apworld` file in your Archipelago `custom_worlds` folder.

## Connecting
1. Launch OpenTTD.
2. Click the **Archipelago** button on the main menu.
3. Enter your server address, slot name, and password.
4. Click **Connect** — the game will generate a world automatically.

## Gameplay
- All vehicles start locked except your starting vehicle.
- Complete missions (shown as goals in-game) to send location checks.
- Receive items from other players to unlock additional vehicles.
- Trap items cause temporary setbacks — beware!

## Win Conditions
Depending on the YAML settings, you may need to reach a target:
- **Company Value** (default): reach target £ value
- **Town Population**: grow a town to target size
- **Vehicle Count**: run that many vehicles simultaneously
- **Monthly Profit**: earn that much in a single month
